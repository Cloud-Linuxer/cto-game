## 애니메이션 효과

### 1. Smooth Transition (모든 구간)

값 변경 시 500ms 동안 부드럽게 전환:

```
신뢰도 50% → 70%

[████████████        ] (시작)
 ↓ 100ms
[█████████████       ]
 ↓ 100ms
[██████████████      ]
 ↓ 100ms
[███████████████     ]
 ↓ 100ms
[████████████████    ]
 ↓ 100ms
[█████████████████   ] (완료)
```

### 2. Pulse Animation (15% 미만)

빨간색 게이지가 깜빡임:

```
[██ ] → [██■] → [██ ] → [██■] (반복)
 밝음    어두움   밝음    어두움
```

### 3. Shimmer Effect (모든 구간)

게이지 바에 빛나는 효과:

```
[████▓▒░        ]
      ↓
[███░▓▒█        ]
      ↓
[████▓▒░        ] (반복)
```

## 실제 사용 사례

### 시나리오 1: 성공적인 게임 진행

```
Turn 1:  [████████████        ] 50% (보통)
         ↓ 좋은 선택
Turn 5:  [████████████████    ] 70% (안정적 신뢰)
         ↓ Series C 투자 유치
Turn 10: [██████████████████  ] 85% (안정적 신뢰)
         ↓ 공격적 확장
Turn 15: [███████████████████ ] 90% (안정적 신뢰)
         → IPO 준비
```

### 시나리오 2: 위기에서 회복

```
Turn 1:  [████████████        ] 50% (보통)
         ↓ 실패한 선택
Turn 3:  [████████            ] 35% (주의 필요)
         ↓ 연속 실패
Turn 5:  [████ PULSE!         ] 12% (즉시 대응!)
         ⚠️ 게임오버 위험!
         ↓ 신뢰도 회복 선택
Turn 7:  [██████              ] 25% (위기 경고)
         ↓ 안정화 노력
Turn 10: [████████████        ] 50% (보통)
         → 회복 성공
```

### 시나리오 3: 게임 오버

```
Turn 1:  [████████████        ] 50% (보통)
         ↓ 잘못된 인프라 선택
Turn 3:  [██████              ] 25% (위기 경고)
         ↓ 용량 초과 사고
Turn 5:  [███ PULSE!          ] 8% (즉시 대응!)
         ⚠️ 마지막 기회!
         ↓ 회복 실패
Turn 6:  [█ GAME OVER!        ] 3%
         💀 서버 장애로 게임 오버
```

## 접근성 고려사항

1. **색상 + 텍스트**: 색맹 사용자를 위해 상태 메시지 병행
2. **명확한 구분**: 임계값 마커로 구간 명확히 표시
3. **충분한 대비**: WCAG AA 기준 충족 (4.5:1 이상)
4. **애니메이션 제어**: prefers-reduced-motion 고려 (향후 개선)
